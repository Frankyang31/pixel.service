# 腾讯云 COS 配置指南

> 本指南教你如何在腾讯云上创建对象存储（COS），用于存储生成的图片和视频。

## 📌 前置要求

- 腾讯云账号（已有）
- 已登录腾讯云控制台
- 已经实名认证

## 🎯 5 分钟快速配置

### 第 1 步：进入 COS 控制面板（1 分钟）

1. 登录腾讯云：https://console.cloud.tencent.com/
2. 在顶部搜索框搜索 **"对象存储"** 或 **"COS"**
3. 点击进入 **"对象存储 COS"** 服务

![进入 COS 控制面板](image-placeholder)

---

### 第 2 步：创建存储桶（Bucket）（2 分钟）

1. 在左侧菜单，点击 **"存储桶列表"**
2. 点击 **"创建存储桶"** 按钮（蓝色大按钮）

**填写以下信息：**

| 字段 | 填写值 | 说明 |
|------|--------|------|
| **存储桶名称** | `pixel-assets` | 自己起个名字，记住它 |
| **地域** | `上海` 或你所在地 | 选择最近的地域 |
| **访问权限** | `私有读写` | 安全起见，改为私有 |
| **存储类型** | `标准存储` | 默认即可 |
| **加密** | `不加密` | 默认即可 |

3. 点击 **"创建"** 按钮

![创建存储桶界面](image-placeholder)

等待 1-2 秒，存储桶创建完成。

---

### 第 3 步：获取 API 密钥（2 分钟）

1. 在腾讯云控制台顶部，点击右上角 **账号头像** → **访问管理**

2. 在左侧菜单，点击 **"访问密钥"** → **"API 密钥管理"**

3. 点击 **"新建密钥"** 按钮

**获取到 2 个密钥：**
- **SecretId** - 类似用户名
- **SecretKey** - 类似密码

⚠️ **重要**：
- **必须保存下来**，密钥只会显示一次
- 不要分享给别人
- 妥善保管

![API 密钥页面](image-placeholder)

```
✅ SecretId：AKID...
✅ SecretKey：...
```

---

## 📝 部署时填入的值

在 `.env.production` 中填入：

```env
# 腾讯云 COS 配置
TENCENT_COS_SECRET_ID=你的_SecretId
TENCENT_COS_SECRET_KEY=你的_SecretKey
TENCENT_COS_BUCKET=pixel-assets
TENCENT_COS_REGION=ap-shanghai  # 根据你选择的地域填写
```

## 🆘 常见问题

### Q1：我在哪里找到存储桶名称？
A：进入 COS 控制面板 → 存储桶列表，可以看到你创建的所有存储桶。

### Q2：我不小心删除了密钥怎么办？
A：进入 API 密钥管理，创建新的密钥即可。

### Q3：文件存在 COS 上安全吗？
A：腾讯云 COS 提供：
- 数据加密存储
- 访问控制（私有读写）
- 自动备份
- 比本地存储更安全

### Q4：创建的 COS 数据会不会被删除？
A：不会。你的数据会持续保存，直到你主动删除。

### Q5：能在国内外使用吗？
A：可以，但国内访问速度最快。
